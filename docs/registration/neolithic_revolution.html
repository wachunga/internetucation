This is a comparatively large lesson because the Neolithic Revolution is such a major event in ancient history. The lesson actually consists of five smaller ones, each discussing a certain aspect of the Revolution. The categories down to which the topic has been broken are: agriculture, geography, organization, society, and technology. These categories should be divided among the available computers as evenly as possible so that an approximately equal number of students have the same assigment.

Regardless of category, each lesson is organized to reflect the steps a historian takes when studying the past. Approximately thirty to thity-five minutes should be allowed for the reading component of the lesson. Students do not necessarily have to read all the data presented if time is limited. The paragraph should only require approximately twenty minutes. Whatever time remains should be used to post the paragraph on the message board and for students to read each other's posts.

Note that printing off the paragraph is optional, but will motivate the students more if they know that the assignment will be marked.
